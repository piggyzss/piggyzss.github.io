一些TS小技巧
2021/01/17

1、keyof

keyof 与 Object.keys 略有相似，只不过 keyof 取 interface 的键

interface Point {
    x: number;
    y: number;
}
// type keys = "x" | "y"
type keys = keyof Point;

栗子：

假设有一个 object 如下所示，我们需要使用 typescript 实现一个 get 函数来获取它的属性值：

const data = {
  id: 1,
  score: 100,
  tag: 'A',
}

function get(o: object, name: string) {
  return o[name]
}

我们刚开始可能会这么写，不过它有很多缺点

无法确认返回类型：这将损失 ts 最大的类型校验功能

无法对 key 做约束：可能会犯拼写错误的问题

这时可以使用 keyof 来加强 get 函数的类型功能：

function get<T extends object, K extends keyof T>(o: T, name: K): T[K] {
  return o[name]
}

2、Partial

Partial 作用是将传入的属性变为可选项

Partial 源码：

type Partial<T> = { [P in keyof T]?: T[P] };

栗子：

interface Params {
  id: number;
  score: number;
  tag: string;
}

// Partial<Params>
// { id?: number; score?: number; tag?: string; }

const initData = (payload: Partial<Params>): Params => merge({
  id: 1,
  score: 100,
  tag: 'A',
})

initData({id: 2})

3、Pick

Pick 的作用是从 T 中取出一系列 K 的属性

Pick 源码：

type Pick<T, K extends keyof T> = { [P in K]: T[P] };

栗子：

// 相当于: type PickParams = { id: number; tag: string; }
type PickParams = Pick<Params, 'id' | 'tag'>

4、Required

Required的作用是将传入的属性变为必选项.

Required 源码：

type Required<T> = { [P in keyof T]-?: T[P] };

5、Readonly

Readonly的作用是将传入的属性变为只读选项

Readonly 源码：

type Readonly<T> = { readonly [P in keyof T]: T[P] };

6、Record

Record的作用是将 K 中所有的属性的值转化为 T 类型

K可以是联合类型、对象、枚举…

Record源码：

type Record<K extends keyof any, T> = { [P in K]: T };

栗子：

type petsGroup = 'dog' | 'cat';
interface PetInfo {
    name:string,
    age:number,
}
type Pets = Record<petsGroup, PetInfo>;

const animalsInfo:Pets = {
    dog:{
        name:'dogName',
        age:2
    },
    cat:{
        name:'catName',
        age:3
    },
}

7、Omit

Omit的作用是未包含

Omit源码

type Omit = Pick<T, Exclude<keyof T, K>>

栗子：

interface Params {
  id: number;
  score: number;
  tag: string;
}
type OmitParams = Omit< Params, "id">
// 相当于: type OmitParams = { score: number; tag: string; }

8、infer

infer的作用是表示在 extends 条件语句中待推断的类型变量

栗子

type ParamType<T> = T extends (param: infer P) => any ? P : T;

如果 T 能赋值给 (param: infer P) => any，则结果是 (param: infer P) => any 类型中的参数 P，否则返回为 T。

interface User {
  name: string;
  age: number;
}

type Func = (user: User) => void;

type Param = ParamType<Func>; // Param = User
type OtherParam = ParamType<string>; // string

9、ReturnType

ReturnType的作用是获取返回类型

ReturnType源码

type ReturnType<T> = T extends (
  ...args: any[]
) => infer R
  ? R
  : any;

栗子

const mapStateToProps = state => ({
  ***
})

export type Props = {
  ***
}
& { dispatch: Dispatch }
& ReturnType<typeof mapStateToProps>
